# Account Security and Authentication Policy

How we handle security requests to protect customer privacy and credentials.

## Password Resets
- Customers can reset their passwords using the "Forgot Password" link on the login page.
- For security reasons, support agents cannot see passwords or set passwords manually over chat or email.

## Two-Factor Authentication (2FA) Issues
- If a customer is locked out of their account due to a lost 2FA device, they must verify their identity.
- Identity verification requires providing:
  1. The registered email address.
  2. The last 4 digits of the card on file or the ID of the last successful order.
- Once verified, support agents can disable 2FA to allow a recovery login.

## Account Lockouts
- Accounts are temporarily locked after **5 failed login attempts** for 5 minutes.
- Lockout resets automatically. Support agents cannot bypass this timer.
